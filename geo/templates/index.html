What is Geomancer?

Geomancer is an open-source tool for journalists to easily add context to existing data, by finding & combining other relevant datasets by location.

To get started, all you need is a dataset that includes geographic information.


Examples of what you can do:

  • Add population data to crime count data, in order to calculate crime rates

Geomancer is funded by a Knight Foundation Prototype Grant
