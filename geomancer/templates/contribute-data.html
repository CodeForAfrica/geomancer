How to contribute data to Geomancer

Do you have a data source you'd like to add to Geomancer? We built this as in an open, extensible platform, so anyone can contribute.

Note: adding a data source requires programming in Python.

Step 1: Fork or clone the Geomancer repo

Step 2: Create a new mancer class

Step 3: Implement the search method

Step 4: Implement the geo_lookup method

Step 5: Register your mancer and test it

Step 6: Submit a pull request!
